# mobile-phone-chat-bot
A simple Python chatbot for a mobile shop that helps customers buy phones, check prices, repair devices, and find accessories. Built using OOP and regular expressions for intent recognition with an interactive command-line interface.
# Mobile Shop ChatBot 

A simple chatbot project built with Python that simulates a mobile shop assistant.
The chatbot can help customers with:

* Buying phones
* Checking phone prices
* Repair services
* Mobile accessories
* General customer support

## Features

* Uses Python OOP concepts (Classes & Objects)
* Uses Regular Expressions (`re`) for intent matching
* Interactive command-line chat
* Supports exit commands like:

  * `quit`
  * `exit`
  * `bye`

## Technologies Used

* Python
* Regular Expressions (`re`)
* Random module

## Example Conversation

Enter your name: Omar
Hello! Omar Welcome to our Mobile Shop

You: I want to buy a phone
Bot: We have Samsung, iPhone and Xiaomi phones available.

You: how much
Bot: Prices start from 3000 EGP depending on the model.

You: bye
Bot: Thank you for visiting our mobile shop. Goodbye!
